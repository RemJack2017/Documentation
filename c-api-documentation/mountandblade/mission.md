# Mission

你可以把Mission看作游戏模式\(比如战役/团队死亡竞技\)之类的东西。Mission与[Scene](../engine/scene.md), [MissionViews](missionbehaviour/missionview.md), 还有 [MissionBehaviours](missionbehaviour/README.md)一起被生成。

关于Mission的更多信息，等我们搞到`CampaignSystem` DLL的访问途径时会放出。

## 提示

* 通过`Mission.Current`属性，你可以获取当前Mission的实例（instance）。（假如它存在的话）
